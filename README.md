# 星芽岛 iPad 网页版 v1.3.2 正式部署版

本版本基于已经通过电脑测试的 v1.3.1 制作，正式移除海底邮局岛。

## 当前岛屿顺序

1. 自然小木屋岛
2. 火星机器人基地岛
3. 椰子岛
4. 雨林岛

## v1.3.2 更新

- 删除海底邮局岛的场景、地图入口、建设配置、奖励文案和全部离线素材。
- 火星机器人基地岛完成后，直接解锁椰子岛。
- 椰子岛完成后，解锁第四座雨林岛。
- 原来停留在海底邮局岛的用户，升级后会自动进入椰子岛。
- 海底邮局岛的历史建设记录会被清理。
- 自然岛、火星机器人岛、椰子岛和雨林岛的已有进度与布局保持不变。
- 继续使用同源分层建设：完成最后一步时不会突然切换成另一张画面。
- 升级离线缓存至 v1.3.2。

## 部署

将压缩包中的全部文件上传到 GitHub Pages 网站根目录。不要额外套一层文件夹。

iPad 已添加到主屏幕时，部署后建议：

1. 完全关闭星芽岛。
2. 在 Safari 中重新打开网站并刷新。
3. 再从主屏幕图标进入。
